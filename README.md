<h2>Installation</h2>

_Below is how install and setting up app._

#### 1. Clone the repo
   ```sh
   git clone https://github.com/karton92/internship-members.git
   ```
#### 2. Install NPM packages
   ```sh
   npm install
   ```
#### 3. Run application (server will also start)
   ```sh
   npm start
   ```
